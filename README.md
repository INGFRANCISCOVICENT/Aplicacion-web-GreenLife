# Green Life 🥗🌱

Green Life es una plataforma web desarrollada en ASP.NET Core (C#) diseñada para la gestión de recetas vegetarianas y la promoción de un estilo de vida saludable. 

## 🚀 Características Principales

* *Gestión de Recetas:* Creación, lectura, actualización y eliminación de recetas completamente vegetarianas.
* *API RESTful:* Exposición de endpoints para integrar la gestión de recetas e ingredientes con otros servicios o aplicaciones front-end.
* *Control de Ingredientes:* Administración detallada de los ingredientes necesarios para cada preparación, enfocándose en alternativas saludables.
* *Arquitectura Organizada:* Proyecto estructurado bajo el patrón de diseño MVC (Modelo-Vista-Controlador).

## 🛠️ Tecnologías Utilizadas

* *Backend:* C# con ASP.NET Core
* *Base de Datos:* SQL Server
* *Arquitectura:* MVC (Model-View-Controller)
* *Control de Versiones:* Git & GitHub

## ⚙️ Requisitos Previos para Desarrollo Local

Para ejecutar y modificar este proyecto en tu máquina local, necesitarás:
